# برنامج المونتاج الاحترافي - نسخة مستقلة (Standalone)

هذه نسخة مستقلة من برنامج المونتاج تم استخراجها من منصة Avinar. يمكن تشغيلها بشكل منفصل تماماً لأغراض التطوير أو الاستخدام المستقل.

## المميزات المتضمنة:
- القص والتقسيم الذكي (Razor Tool).
- التدوير والتحريك المباشر على منطقة العمل (Rotation & Transform).
- إضافة النصوص والتحكم في خصائصها.
- معالجة الألوان والسينما (Color Grading).
- التراجع والإعادة (Undo/Redo).
- المساعد الذكي AI (يتطلب اتصالاً بالإنترنت والـ API مفعل).

## طريقة التشغيل:

1. تأكد من وجود **Node.js** مثبتاً على جهازك.
2. قم بفتح مجلد `standalone_editor` في محرر الأكواد (VS Code) أو في Terminal.
3. قم بتشغيل الأمر التالي لتثبيت المكتبات المطلوبة:
   ```bash
   npm install
   ```
4. بعد اكتمال التثبيت، قم بتشغيل البرنامج:
   ```bash
   npm run dev
   ```
5. افتح المتصفح على الرابط الذي سيظهر لك (عادة ما يكون `http://localhost:8080`).

## ملاحظة تقنية:
تعتمد هذه النسخة على نفس ملف `package.json` الأساسي لضمان توفر كافة المكتبات (Lucide, Radix UI, Tailwind CSS, etc).
